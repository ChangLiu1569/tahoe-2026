# Tahoe '26 — Trip Dossier 🏔️🛶

A phone-first trip planner for a Lake Tahoe weekend (July 3–5, 2026).

**Live site:** https://changliu1569.github.io/tahoe-2026/

A single self-contained `index.html` — open it on any phone or computer. Tabs for:

- **Plan** — trip summary + a swipeable 3-day itinerary
- **Map** — interactive map of every spot, with one-tap directions
- **Food** — Sacramento lunch stops + Tahoe groceries
- **Lake** — paddle board rentals, beaches & hikes, July 4th fireworks
- **Pack** — a checklist that saves your progress on your device

Built as a static page (no build step). The map uses Leaflet + OpenStreetMap; everything else works offline.
